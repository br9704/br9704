### What I've Learned Building These

**AI integration in production apps.** Sentinel uses Gemini 2.5 Flash for claim verification across multiple sources. Getting reliable, structured output from LLMs at scale taught me more about error handling and retry logic than any course.

**Real-time architecture.** The Collab Dashboard uses WebSockets for live sync between users. Handling reconnection, state reconciliation, and optimistic UI updates was the hardest part of building it.

**3D on the web.** The GitHub Visualizer renders contribution data as interactive 3D geometry with Three.js. I learned camera controls, raycasting, and keeping 60fps with thousands of objects.

**Shipping solo.** Sentinel is a full-stack iOS app with Supabase backend, Edge Functions in Deno, and real-time data pipelines. Building it alone forced me to make architectural decisions fast and own every layer of the stack.
